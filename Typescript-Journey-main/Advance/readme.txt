

create an environment with 2 folder for source(src) and distination(dist)
- src for storing typescript files
- dist for string javascript files

Process :

    - create tsconfig file using command "tsc --init"
    - create a package.json file using command "npm init-y"
    - create live server using command "npm i lite-server"
    - run in watch mode for create typescript file to javascript file in dist folder using command "tsc -w"
    - explore the file file and change output dir to ./dist for creating js file in dist folder
    - follow the code
